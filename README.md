# Customer-Sentiment-Analysis-
A full-stack web application that analyzes customer feedback using Natural Language Processing (NLP), categorizing sentiment, emotions, urgency, and more — with support for text input and file uploads.

---

## 🌐 Live Demo

https://capitol-olympus-programmers-brighton.trycloudflare.com/

> ⚠️ Note: This is a temporary live demo. The link may change when the server restarts.

---

## 📌 Features

* 🔍 **Sentiment Analysis** (Positive, Negative, Neutral)
* 😊 **Emotion Detection** (Joy, Anger, Sadness, etc.)
* 📂 **File Upload Support**

  * CSV
  * Excel
  * PDF
* 🧠 **Custom NLP Enhancements**

  * Extended sentiment lexicon
* 🗂️ **Feedback Categorization**

  * Product
  * Service
  * Delivery
* ⚡ **Urgency Detection**

  * Critical / High / Normal
* 🧾 **History Tracking**

  * Stores previous analysis runs
* 📊 **Trend Analysis**

  * Visualize sentiment over time
* 📥 **Export Options**

  * CSV download
  * PDF reports

---

## 🛠️ Tech Stack

* **Backend:** Flask
* **Database:** SQLite (via SQLAlchemy)
* **NLP:** NLTK (VADER Sentiment Analyzer)
* **Data Processing:** Pandas
* **File Handling:** PyPDF, CSV
* **Reporting:** ReportLab
